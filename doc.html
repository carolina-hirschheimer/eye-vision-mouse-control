About Mouseye

Mouseye is an application used to help people with disabilities and limited body motion control their computer mouse through their eyes. The application captures eye movement to move the mouse across the screen, as well as different blinks for a right or left click.

Configuration

After installing the required dependencies, you can execute the program in 4 different modes, displayed by the options below:

Mode Functionality Argument
Calibration Used to calibrate the corners of the page. In order to do so, you need to look at one of the 4 corners of the computer window and click on the corresponding calibration button in the application interface. calibrate
Mouse Control Only the mouse control is enabled, without the clicks. In order to move the mouse, you just need to look through the page at where you want the mouse to move to. control_mouse
Click Control Only the click control in enabled, and the mouse control is still manual. In order to left-click the mouse, you need to blink with only one eye. In order to right-click the mouse, you need to blink with both eyes. It is important that you hold on to the blink for a couple of seconds, so that the application does not misidentify regular blinks as clicks. control_click
Both Controls Both the mouse and the click controls are enabled to the user according to the specifications above. control_both

  • mac

Then, to start the application, go to the /source/project/ folder and run the main.py file, replacing the "[argument]" by one of the arguments presented in the table of the Configuration section based on the functionality you want to use. 

              
                  $ cd source/project/ 

                  $ python main.py [argument]
